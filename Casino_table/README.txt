Что бы добавить "Casino_table" (КТ) модуль, необходимо:
1) в папке с темой создать папку template_parts в которую поместить casino_table.php + папку с картинками casino_table_img
2) в то место, где Вы хотите добавить вывод данной таблици в *.php файле надо создать вызов <?php  get_template_part('template_parts/casino_table');  ?> 
3) на соответствующией странице должен был отобразиться данный модуль с двумя примерами. 
4) Подключить в свой корнивой фаил со скриптами код (если его нет, после jQuery и после формирования DOM дерева);
 
function likeLink(btn) {
        btn.on("click", function(){
            var urlLink = $(this).attr("data-href");
            if ( urlLink === "" ){
                alert("URL address isn't correct / шляпа" );
            }else{
                window.open(urlLink);
            }
        })
}

  likeLink($(".tb_casino__logo"));
  likeLink($(".btn_casino__play"));

/*Данный скрипт обеспечит срабатывали ссылки заданые в data-href. URL адресса ссылок писать полностью! */

	!Картинки одинакового смысла делать одинаковых форматов и разрешений
	!Картинки казино находящиеся в папке  casino_table_img\casinos предпочтительно задавать размерами 320х245px
	
6) Назначаем alt и data-href всем необходимым элементам
7) Оптимизируем фотографии, сжимаем css
8) После окончания модификаций переносим стили из casino_table.php в основной фаил стилей темы
